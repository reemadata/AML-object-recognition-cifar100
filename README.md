# Object Recognition using CIFAR-100 Dataset

## Overview

This project focuses on object recognition and image classification using the CIFAR-100 dataset.

The objective is to classify images into object categories by extracting visual features and applying machine learning and deep learning techniques.

---

## Dataset

CIFAR-100 contains:

- 100 object categories
- 50,000 training images
- 10,000 testing images
- RGB images of size 32×32

---

## Project Pipeline

### 1. Image Preprocessing

- Image normalization
- Feature extraction
- Data preparation

### 2. Feature Engineering

- Histogram of Oriented Gradients (HOG)
- Principal Component Analysis (PCA)

### 3. Classification Models

Implemented models include:

- Support Vector Machine (SVM)
- Neural Network
- Convolutional Neural Network (CNN)

### 4. Evaluation

Performance was evaluated using:

- Accuracy
- Confusion Matrix
- Class-wise Recognition Rate

---

## Results

The experimental results demonstrate the effectiveness of machine learning techniques for object recognition on CIFAR-100 images.

Performance comparison between multiple classifiers was conducted to identify the most suitable approach.
